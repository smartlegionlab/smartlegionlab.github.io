2FA Manager Ecosystem

The Smart 2FA ecosystem is a cross-platform offline TOTP management suite — featuring CLI (Python/Bash), Desktop GUI (PyQt5), Android app, and a secure backend library — all using AES-256-GCM encrypted local storage, GPG-backed backups, QR code import/export, and zero cloud dependency for generating time-based one-time passwords.
