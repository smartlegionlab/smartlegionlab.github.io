GitHub Use Bot

Asynchronous task manager. Use Telegram bot.

TODO app (task management app) is a simple app that allows users to create, edit, delete and track their tasks. This bot can be deployed even on a phone using Termux.

  • Desktop ToDo app
  • CLI ToDo app
  • Web ToDo app

  • Architectural code refactoring
  • Rewritten using MVC (Model-View-Controller) architecture
  • Bugs fixed
  • User interface simplified and improved
  • Code structure and readability improved (SOLID, DRY, MVC)

In the next updates:

  • Adding comments to tasks
  • Adding subtasks to tasks

MariaDB Setup

sudo mysql -u root -p
CREATE USER 'USER_NAME'@'localhost' IDENTIFIED BY 'PASSWORD';
CREATE DATABASE database_name CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
